Usage: notebooklm-mcp [OPTIONS] COMMAND [ARGS]...

  NotebookLM MCP Server - Professional automation for Google NotebookLM

Options:
  --version          Show the version and exit.
  -c, --config PATH  Configuration file path
  --debug            Enable debug logging
  --help             Show this message and exit.

Commands:
  chat            Interactive chat with NotebookLM
  config-show     Show current configuration
  export-profile  Export Chrome profile for sharing
  import-profile  Import existing Chrome profile
  init            Initialize NotebookLM MCP Server with notebook URL
  quick-setup     Quick setup with config file and optional profile import
  server          Start the FastMCP v2 NotebookLM server
  test            Test connection to NotebookLM
